# Melon_Music_Award
MelonMusicAward Data Analysis

멜론뮤직어워드(MMA) 수상자와 해당연도 차트데이터를 분석하기 위한 프로젝트

# 심사기준(공식)
**2024.10.31~2025.11.19** 발매된 모든 음원 각 부문별 음원 점수 집계  
단, 트랙제로 초이스 부문은 심사 기간 상이하고 핫트렌드 부문은 발매 시기 제약 없음
